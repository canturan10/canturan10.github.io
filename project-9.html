A system of Call Center Activities Supported by AI

Project Details

In this project, it is aimed to automate the performance of customer representatives by using natural language processing (NLP) and machine learning system that provides analysis of daily spoken language by working with current artificial intelligence technologies, whose importance has recently increased.

This system, which works with the deep learning mechanism, will determine the performance of the operators by obtaining the appropriate data set for the language model and using this data set with the trained model after the speeches in the call centers are translated into text.
